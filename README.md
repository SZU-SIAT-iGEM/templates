# templates
作为SZU-SIAT-iGEM2026的模板仓库，存放各大仓库所需的提交模板。
## 📁 templates

### 用途说明

存放 **所有模板文件**，供团队成员复制使用。

### 目录结构

```
templates/
├── 📄 月度报告模板.md
├── 📄 周报模板.md
├── 📄 会议记录模板.md
├── 📄 Protocol模板.md
├── 📄 实验数据README模板.md
├── 📄 实验日志模板.md
├── 📄 访谈记录模板.md
├── 📄 视频索引模板.txt
├── 📁 issue模板/
│   ├── 实验计划.md
│   ├── 通用任务.md
│   ├── Bug报告.md
│   └── 讨论议题.md
└── 📄 README.md
```

### 使用方法

1. 找到需要的模板
2. 复制模板内容
3. 在目标位置创建新文件
4. 粘贴并按要求填写
5. 删除不需要的提示文字

### 模板更新

如需更新模板：
1. 创建 Issue 说明更新理由
2. 提交修改后的模板
3. 在 Commit 信息中说明变更

## 📖 使用方法

### 方法一：复制粘贴（推荐）

1. 打开需要的模板文件
2. 点击查看原始内容
3. 全选复制 (Ctrl+A, Ctrl+C)
4. 在目标位置创建新文件
5. 粘贴内容并按需修改

### 方法二：下载后使用

1. 点击模板文件
2. 点击 **Download** 按钮
3. 在本地编辑后上传

---

## ⚠️ 注意事项

1. **保持模板原样** - 请勿直接修改模板文件
2. **删除提示文字** - 使用时删除 `<!-- -->` 中的提示
3. **必填项** - 标注为必填的项目请务必填写
4. **更新模板** - 如需修改模板，请先创建 Issue 讨论
